# Fourier Optics Simulations

This repository explores **Fourier Optics**, a framework that describes light as waves and their frequency components. It provides intuitive ways to study how light propagates, diffracts, and forms images. Using this approach, you can:

- Understand diffraction and interference patterns  
- Predict image formation through lenses and apertures  
- Reconstruct optical fields from measured diffraction patterns  

By working in the frequency domain, complex wave interactions become easier to analyze, making this approach a powerful tool for microscopy, photonics, and optical engineering.


## 🌟 Applications

- Simulating diffraction from apertures and gratings  
- Designing optical imaging systems  
- 3-d imaging and reconstruction using Holography
