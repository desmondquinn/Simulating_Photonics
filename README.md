# 🌐 Simulating Photonics

## 📌 Overview

This repository explores a wide range of concepts in **photonics and optical physics** through numerical simulations. It brings together multiple computational approaches to model how light interacts with materials, structures, and devices.

The goal of this project is to build physical intuition by implementing simulations from first principles as well as using established numerical methods.

## 🔬 Topics Covered

This repository includes simulations across several key areas of photonics:

- **Fourier Optics**  
  Wave propagation, diffraction, and imaging using frequency-domain methods  

- **Mie Scattering**  
  Light scattering from dielectric and metallic nanoparticles  

- **Thin Film Optics (TMM)**  
  Reflection, transmission, and absorption in layered media  

- **FDTD Simulations**  
  Time-domain modeling of electromagnetic wave propagation  

- **Photonic Crystals & Waveguides**  
  Light confinement and guiding using periodic dielectric structures  

---

## 🧠 Scientific Motivation

Photonics is fundamentally about understanding light as a wave. Many optical phenomena—such as diffraction, interference, and resonance—can be complex to visualize.

This repository demonstrates how:

- Complex wave behavior can be modeled numerically  
- Physical systems can be understood through simulation  
- Analytical concepts translate into computational tools  

By combining theory with simulation, these examples provide deeper insight into optical systems used in modern technologies.

## 📂 Repository Structure

- `Fourier_optics/` → Diffraction and wave propagation (Angular Spectrum Method)  
- `Mie_code/` → Scattering from particles and multipole decomposition  
- `Thin_film_optics/` → Transfer Matrix Method simulations  
- `FDTD_experiments/` → Custom electromagnetic simulations  
- `Photonic_crystals/` → Waveguides and periodic structures  

---

## ⚙️ Tools & Libraries

The simulations are implemented in Python using:

- `numpy` – numerical computations  
- `matplotlib` – visualization  
- `scipy` – special functions and physics tools  
- `tmm` – thin film optics  
- `meep` – FDTD simulations
